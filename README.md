##DWON

http://maven.nlpcn.org/org/nlpcn/nlp-lang/


#### 这个项目是一个基本包.封装了大多数nlp项目中常用工具

> 工具
* √ 词语标准化
* √ trie树结构
* √ 双数组trie树
* √ 文本断句
* √ html标签清理



> 组件
* √ 汉字转拼音
* √ 简繁体转换
* √ bloomfilter
* √ 指纹去重 
* √ 词共现统计
* √ 基于内存的搜索提示
